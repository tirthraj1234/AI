Learn the K-Nearest Neighbors (KNN) algorithm, understand how it works, implement it using Python and Scikit-learn, and build an Employee Attrition Prediction project.

## Topics Covered

- Introduction to KNN
- Distance Metrics
- Choosing the Value of K
- Lazy Learning
- Feature Scaling
- KNN Hyperparameters
- KNN Implementation
- Employee Attrition Prediction
- Model Evaluation

## Project

Employee Attrition Prediction using KNN

### Features

- Data Loading
- Data Preprocessing
- Feature Scaling
- Model Training
- Prediction
- Accuracy Evaluation
- Confusion Matrix
- Classification Report

## Technologies Used

- Python
- Pandas
- Scikit-learn
