# Day 8 – Clustering

## Objective

Learn the fundamentals of Unsupervised Machine Learning using clustering techniques and build a Customer Segmentation project with K-Means Clustering.

## Topics Covered

- Introduction to Clustering
- Types of Clustering
- K-Means Clustering
- Centroids
- Euclidean Distance
- Elbow Method
- WCSS (Within-Cluster Sum of Squares)
- Silhouette Score
- K-Means Hyperparameters
- Model Evaluation

## Projects

### 1. K-Means Implementation
- Dataset loading
- Feature selection
- Model training
- Cluster assignment
- Centroid visualization

### 2. Customer Segmentation
- Customer dataset preprocessing
- Elbow Method
- K-Means training
- Cluster visualization
- Business interpretation

## Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-learn

## Learning Outcome

- Understood clustering concepts.
- Learned different clustering techniques.
- Implemented K-Means using Scikit-learn.
- Determined the optimal number of clusters.
- Built a Customer Segmentation project.
- Evaluated clustering quality using WCSS and Silhouette Score.
